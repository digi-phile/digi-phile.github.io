Comparison of Two Different Types of Linear Regression Models

16/07/18 19:46

Abstract



Regression is a subtopic of machine learning, which specializes on examining the relationships between inputs and outputs as a powerful statistical tool.



There are many different types of regression methods, including linear regression, logistic regression, etc, but all of which have the core of determination of correlations between independent variables and dependent variables.



Linear regression is which this demonstration focuses on.



General Principle of Linear Regression



There are two general categories of linear regression: Simple linear regression and Multiple linear regression. The way they differ is that simple linear regression takes a single predictor variable input x and multiple linear regression takes multiple and/or vector predictor variable input X.



Intuition of Simple Linear Regression



In the context of a simple linear model, we assume that Y is linked to X via the regression coefficients α, β. As simple linear regression only have a single independent variable as input and a corresponding dependent variable as output, it can be expressed in a way of a straight line with β being the gradient and α being the y-intercept, plus ε being the omnipresent random error.



The application of simple linear regression is broad. For instance, it works sufficiently well with predicting yield based on the rainfall because the nature that other factors do not have correlation with yield that is as deterministic as rainfall, considering yield increases proportionally as rainfall increases.



Relationship between two variables is said to be deterministic if one variable can be accurately expressed by the other. For example, using temperature in degree Celsius is possible to accurately predict Fahrenheit.



Basic mechanism of Simple Linear Regression



The fundamental mechanism of simple linear regression is to obtain a line of best fit, where sum of squares of the distances between every true data value and prediction is minimized, meaning the error of prediction is the smallest.



Mathematically speaking, the correlation can be expressed as:



Y = α + β * x + ε


There are both certain values of α and β where the regression model can be optimized. The sum of squared error can be noted as:



Error = Σ (predicted_value - actual_value) ** 2
