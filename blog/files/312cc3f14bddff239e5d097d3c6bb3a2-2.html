Comparison of Two Different Types of Linear Regression Models

11/10/19 00:08

Abstract



Regression is a subtopic of machine learning, which specializes on examining the relationships between inputs and outputs as a powerful statistical tool.



There are many different types of regression methods, including linear regression, logistic regression, etc, but all of which have the core of determination of correlations between independent variables and dependent variables.



Linear regression and Logistic Regression are which this demonstration focuses on. Perceivably, the only way in which they differ is that Linear Regression uses Continuous variables as output, while Logistic Regression uses Categorical variables as output.



General Principle of Linear Regression



The continuous output which a linear regression model gives means infinite number of values within a given range.



There are two general categories of linear regression: Simple linear regression and Multiple linear regression. The way they differ is that simple linear regression takes a single predictor variable input x and multiple linear regression takes multiple and/or vector predictor variable input X. However, the ideology behind them are essentially the same



In the context of a simple linear model, we assume that Y is linked to X via the regression coefficients α, β, where α is called the bias and β is called the weight. As simple linear regression only have a single independent variable as input and a corresponding dependent variable as output, it can be expressed in a way of a straight line with β being the gradient and α being the y-intercept, plus ε being the omnipresent random error. On the other hand, in the context of a multiple linear model, we assume that all the variables have have weights of β1, β2, β3… plus bias α and random error ε, which are the same as the ones used in simple linear regression model.



The application of linear regression is broad. For instance, it works well with predicting yield based on factors like rainfall, soil acidity, etc because of the nature that all the factors do have correlation with yield that are as deterministic as rainfall. For instance, consider yield increases proportionally within a range as amount of fertilizer applied increases.



Relationship between variables is said to be deterministic if one variable can be accurately expressed by the other. For example, using temperature in degree Celsius is possible to accurately predict Fahrenheit for linear regression. However, this can be a bad example for linear regression because the data can be overfitted by the model which means the model works perfect for the train dataset but does not work for the test dataset. This applies to logistic regression as well.



Basic mechanism of Linear Regression



To simplify the ideology of linear regression, let us look at simple linear regression instead of using many variables as input because it is fairly easy to understand and demostratable on a 2D plane. The fundamental mechanism of linear regression is to obtain a line of best fit, where sum of squares of the distances between every true data value and prediction is minimized, meaning the error of prediction is the smallest.



Mathematically speaking, the correlation can be expressed as:



Y = α + β * x + ε


As a way to assess the performance of the regression model, different models are to be tested for the optimal result. But for a particular model, there are both certain values of α and β where the regression model can be optimized. The sum of squared error can be noted as:



Error = Σ (predicted_value - actual_value) ** 2


Code Demonstration of Linear Regression



Let us take a real world example of linear regression where we predict prices of second-hand iPads based on variables including sold or not, storage, condition, carriers, etc. First thing first, we import our data from a CSV file and split the dataset into two: train data & test data with a ratio of 70% to prevent the prediction from Overfitting, meaning the train data is over-trained and outperform the test data, leading to performance failure.



Ebay_data = read.csv('eBayClean.csv')

set.seed(100)
split = sample.split(Ebay_data$sold, SplitRatio = 0.7)
train = Ebay_data[split,]
test = Ebay_data[!split,]


The second step is to actually sit the dataset into the model, with condition, storage and color as the inputs and price as the output. In such a case, it is a Multiple Linear Regression model. From the very last second line, it indicates the R-square value, a statistical measure of fit that indicates how much variation of a dependent variable is explained by the independent variable(s) in a regression model, higher is better. In this case, we have a pretty decent R-square value of 0.8763, which can be interpreted as 88% of the predictions are can be justified by the model.



Ebay_model = lm(startprice~condition+storage+color, data = train)

General Principle of Logistic Regression



The categorical variable a logistic regression gives means limited number of distinct categories. In another word, the outcome is either True or false.



Logistic Regression simply works with probability, predicting whether one event happens or not. This is why logistic regression is sometimes also called binary classification.



There are a lot of examples of logistic regression model. Consider the scenario of the Titanic, where you have variables of personal information about the attendants to predict the survival rate. The output we are predicting is either true, meaning the person survived, or false, meaning the person did not.



Basic Mechanism of Logistic Regression



The procedure in which the output is calculated is essentially the same as the one used in linear regression. However, the output of a logistic regression model goes through mathematical function called logistic function, which is also called the sigmoid function. Whatever real value goes into the function will be converted in to a probability value ranged from 0 to 1, generating a value possibility indicating the likelihood one event is going to be true.



The logistic regression model has correlations for variables as well. But when applying the correlation to other variables, the output has not passed through the sigmoid function yet, so we need to generate the possibility manually.

Besides assessing the accuracy of the predictions, there is another way of measuring the performance of the model comprehensive. Just like what we did for linear regression model, now we are summarizing the model using inbuilt summary function to give out the correlations and performance measurer called AIC. R-Square to Linear Regression is as if AIC to Logistic Regression, higher is better. This index is specific to one dataset, which means it is meaningless when comparing two datasets.
